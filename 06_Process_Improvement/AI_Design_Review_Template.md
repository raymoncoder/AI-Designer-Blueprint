# 🔍 AI-Assisted Design Review Template
*Framework for automated design critiques*

**How to use:** Paste this template into your AI assistant (e.g., Gemini 3.1 Pro) along with a screenshot, JSON, or code of your UI component. 

**Prompt Template:**
> "Act as a Lead Product Designer. I am providing you with a UI component. Please conduct a comprehensive design review using the following criteria, and score each section out of 10:"
> 
> 1. **Visual Hierarchy:** Is the primary call-to-action immediately obvious? Is the eye guided logically through the information?
> 2. **Cognitive Load:** Are there too many elements competing for attention? How can we reduce friction?
> 3. **Edge Cases:** What happens if the user's name is 50 characters long? What if the data fails to load? What if it's viewed on an iPhone SE?
> 4. **Micro-interactions:** Are there opportunities to add delight through subtle hover states or loading animations?
> 
> **Output:** Provide specific, actionable bullet points for improvement.
