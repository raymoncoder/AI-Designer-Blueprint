# ✅ Part 5: The Ultimate Daily Checklist

**Phase 1: Discovery & Research**
- [ ] Feed raw interview transcripts into AI to extract pain points and quotes.
- [ ] Run competitor analysis: Ask AI to summarize the UX flows of 3 direct competitors.
- [ ] Generate 3 divergent user personas using specific demographic constraints.

**Phase 2: Ideation & Wireframing**
- [ ] Use AI to generate an exhaustive list of edge cases for the main flow (e.g., "What if the user's internet drops here?").
- [ ] Generate component structures via v0.dev to visualize layout options.
- [ ] Apply AI auto-renaming and organization to keep Figma layers clean.

**Phase 3: High-Fidelity & UX Writing**
- [ ] Invoke your "UX Writer Skill" to review and rewrite all interface copy for clarity and tone.
- [ ] Generate dynamic empty states, error messages, and success toasts.
- [ ] Use generative AI to create non-generic, high-quality placeholder imagery.

**Phase 4: Handoff & QA**
- [ ] Generate a markdown documentation of the design decisions for the dev team.
- [ ] Run an AI accessibility audit on the final screens (color contrast, tap targets, aria-labels).
- [ ] Create testing scripts for QA based on the user flows.
